-User dashboard
-Admin Dashboard
-Blog system
-Contact page
-Community support

Tech stack:
Frontend:
HTML
CSS
Javascript

Backend:
Python Flask

Database:
Mysql

Payment:
Upi (Google Pay)

Purpose:
To help people share their emotions and feel better by talking with supportive listeners.
